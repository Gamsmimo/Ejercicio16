Proyecto 16: Desplegable Personalizado

Este proyecto implementa un menú desplegable completamente personalizado usando HTML, CSS y JavaScript. El desplegable permite a los usuarios seleccionar un elemento de una lista, con estados predeterminado, abierto y seleccionado.

Características implementadas

  • Estado predeterminado: Muestra el texto "Selecciona un elemento" con una flecha hacia abajo.
  • Estado abierto: Al hacer clic, se despliegan todas las opciones y la flecha cambia de dirección.
  • Estado seleccionado: Muestra el elemento elegido y lo resalta en la lista con un símbolo de verificación.
  • Cierre automático: Al seleccionar un elemento, el desplegable se cierra automáticamente.
  • Cierre al hacer clic fuera: El desplegable se cierra si se hace clic en cualquier parte fuera de él.
  • Interacción con teclado: Se puede navegar por las opciones usando las teclas de flecha y seleccionar con Enter.
